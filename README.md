# discordbot

DiscordBot for the Lazer Krew


This repo was created following https://discordjs.guide/


1. Clone repo
2. ```npm install discord.js```
3. Follow the guide to set up yoru bot with your server (tokens and privileges)
4. ```node .``` to start bot
